# Desafio DIO - Classificar Herói

Script em Node.js que solicita o nome e a quantidade de experiência (XP) de um herói e exibe o nível correspondente.

## Como executar ⚙️

Pré-requisitos:
- Node.js instalado (recomendado v12+)

No diretório do projeto, rode:

```bash
node main.js
```

Digite o `nome` e o `XP` quando solicitado.

## Exemplo de saída

```
Digite seu nome, bravo herói: Arthur
Informe sua quantidade de experiência (XP): 5500
O Herói de nome Arthur está no nível de Ouro!
```

## Observações
- O projeto não depende de pacotes externos (usa `readline` nativo).
- Arquivo principal: `main.js`.
